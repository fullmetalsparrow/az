### Managed Disk Snapshots
	A read-only fully copy of a managed disk that is stored as a standard managed disk by default.

- Managed disk snapshots exist independent of the source disk and can be used to create new managed disks.
- Managed disk snapshots are billed on a consumption basis.
- Managed disk snapshots supprot disk sizes up to 32 TB.
- Managed disk snapshot retention policy is 2 days by default.
	- *Retention policy can be configured between 1-5 days.*
