- Log search rule:
	1. *Kusto query.*
	2. *Time period.*
	3. *Frequency.*
	4. *Threshold.*
*Log search rules return the number of records or a metric measurement.*

- Metric measurement:
	1. *Aggregate function.*
	2. *Group field.*
	3. *Interval.*
	4. *Threshold.*

- Alert rule:
	1. *Resource.*
	2. *Conditions.*
	3. *Actions.*
	4. *Alert details.*
*Log alerts are stateless and will trigger every time the criteria is met.*
